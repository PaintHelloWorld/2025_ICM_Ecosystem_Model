## 📋 项目简介

本项目是针对2025年交叉学科建模竞赛（ICM）Problem E的完整解决方案，开发了一个**森林到农田转变的农业生态系统动态模型**。模型模拟了生态系统在30年内从高强度化学农业逐步过渡到有机农业的过程，并评估了不同管理策略在生态、经济与可持续性三个维度的表现。

### 🎯 核心目标

- 建立包含作物、杂草、害虫、鸟类、蝙蝠和土壤等关键组分的生态系统动态模型。
- 模拟化学农药停用、天敌引入、有机肥料使用等人类决策的长期影响。
- 从生态稳定性、经济效益和系统可持续性三个维度对6种不同农业情景进行综合评估。
- 为农户和政策制定者提供向可持续农业转型的量化依据和建议。

## 🚀 快速开始

### 环境配置

1. 确保安装 Python 3.8+
2. 安装依赖包：

```bash
pip install -r requirements.txt
```

### 运行模拟

进入源代码目录并运行主程序：

```bash
cd Src
python main.py
```

程序将依次执行：
1. **运行6种农业情景的30年模拟**
2. **生成情景对比图**（每个情景单独子图）
3. **生成变量对比图**（同一变量在不同情景下的变化）
4. **生成综合评估柱状图**
5. **在控制台输出最终年份的汇总表格**

## 🧩 模型架构

### 1. 农业生态系统核心模型 (`ecosystem.py`)
- **种群动态**：基于Logistic增长与种间竞争（作物vs杂草，害虫vs天敌）
- **化学过程**：农药/除草剂残留衰减及其对靶标和非靶标生物的影响
- **农业操作**：季节性施肥、施药、有机杂草控制
- **自然过程**：鸟类与蝙蝠的引入、种群恢复与捕食效应
- **土壤系统**：养分消耗与健康度变化（受化肥/有机肥影响）

### 2. 多维情景评估模块 (`evaluation.py`)

- **生态维度**：生产力稳定性、生物多样性、病虫害控制、土壤健康、化学残留
- **经济维度**：平均产量、成本效率
- **可持续性维度**：土壤趋势、系统韧性、长期生产力、均衡性
- **综合评分**：加权计算总体得分（0-100）
- **文字摘要**：自动生成系统状态描述

### 3. 结果可视化模块 (`visualization.py`)
- `plot_scenario_comparison`：6情景并排对比（多变量时间序列）
- `plot_all_scenarios_together`：同变量跨情景对比（6变量分图）

### 4. 主程序 (`main.py`)

- 定义了6种从“高化学投入”到“完全有机”的渐变情景
- 协调30年（120季度）的模拟循环
- 收集并传递模拟历史数据

## 📊 主要特性

### ✅ 模拟情景

| 情景ID    | 名称                   | 化学使用  | 天敌引入 | 化肥类型 | 覆盖作物 | 过渡策略 |
| --------- | ---------------------- | --------- | -------- | -------- | -------- | -------- |
| scenario1 | 常规农业（高化学投入） | 持续使用  | 无       | 化肥     | 否       | 基线     |
| scenario2 | 仅停用化学（无天敌）   | 第2-3年停 | 无       | 化肥     | 否       | 化学退出 |
| scenario3 | 化学退出 + 引入蝙蝠    | 第2-3年停 | 蝙蝠     | 化肥     | 是       | 天敌辅助 |
| scenario4 | 化学退出 + 引入鸟类    | 第2-3年停 | 鸟类     | 化肥     | 是       | 天敌辅助 |

## 🔬 模型特点

### 科学基础
- **种间关系**：基于经典Lotka-Volterra竞争与捕食模型
- **化学动力学**：一级衰减模拟农药残留
- **农业操作**：按季节触发，模拟真实农事日历
- **土壤反馈**：有机/化肥对土壤健康的差异化影响

### 政策与决策支持
- 量化了“化学退出 + 天敌引入”过渡路径的可行性
- 对比了“渐进过渡”与“激进有机”的长短期效益
- 提供了生态、经济、可持续性三重底线分析框架
